# development sources about GaduGadu Protocol:
When creating **OpenGaduServer**, this project uses references from community-made 
libraries, projects, uses software; and simillar things to that. Sources listed below are 
the ones which are most used when troubleshooting and/or implementing functions

### LibGadu _by Wojtekka and LibGadu Team[^1]_: <br>
Explains most of the core functions of the way that client works. The Website version is 
focused on versions from 6.0 and higher - Website: https://libgadu.github.io/protocol<br> 
- Github: https://github.com/wojtekka/libgadu<br>
### Uniksowy Serwer Gadu _by Wojtekka:_ <br>
Is a ruff server implementation logic, it helps with cleaning up the code and getting a 
proper understanding of how is server suppoused to work and communicate with the client. 
- Github: https://github.com/wojtekka/usg<br>
### Eksperymentalny Klient Gadu (EKG) _by EKG team[^1]_
Original source where GaduGadu Protocol was reverse engineered, but became a separate 
project later. - Website: http://ekg.chmurka.net/download.php - Github: 
https://github.com/ekg2/ekg2/
### Wireshark<br>
Best and most advanced to my little knowledge software for inspecting traffic of various 
types - including Gadu-Gadu's HTTP and packets sent from/to server. - Website: 
https://www.wireshark.org/download.html

[^1]: Summing up people that worked on mentioned repository. You can find contributors in 
the `AUTHORS` file in repositories
